# Windows Store Asset Placeholders

Provide the following PNG files before building an MSIX package:

- StoreLogo.png (50x50)
- Square44x44Logo.png (44x44)
- Square150x150Logo.png (150x150)
- Wide310x150Logo.png (310x150)
- SplashScreen.png (620x300)

These names are referenced by AppxManifest.template.xml.
